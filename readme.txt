== Description ==

This plugin has been deprecated and won't be maintained anymore. In the future this plugin might be resurected with a better code base.
Twitter is a great way to grow your websites traffic. With this plugin you can display an lightbox with a twitter likebox. The result? More twitter followers. 

***Features***

*   Display the lighthox on pages,on post and/or on your homepage
*   Set a delay for the lightbox to show
*   Grow you twitter follower base 
*   More exposure by the friends of your new followers
*   Ability to show the lightbox once every x days (per visitor)

[youtube http://www.youtube.com/watch?v=qV5Q01jzKuU]

***Requirement***

*   PHP5
*   Valid Doctype in your theme (usually the case)

== Installation ==

Extract the zip/rar file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
